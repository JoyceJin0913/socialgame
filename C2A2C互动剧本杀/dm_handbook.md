## 🎯 DM权限与职责

### 你拥有的权力
| 权力 | 说明 |
|------|------|
| **NPC控制** | 可操控任何非玩家角色的言行 |
| **事件触发** | 可手动触发预设/随机事件 |
| **难度调整** | 可微调检定成功率（±10%） |
| **时间控制** | 可延长/缩短阶段时长 |
| **仲裁权** | 对争议事项拥有最终裁决权 |
| **信息保密** | 可向特定玩家隐藏/透露信息 |

### 你的核心职责
1. **节奏把控** — 确保游戏流畅不拖沓
2. **公平公正** — 不偏袒任何玩家/阵营
3. **沉浸维护** — 及时纠正OOC行为
4. **冲突调解** — 化解玩家间的真实矛盾
5. **技术保障** - 处理AI引擎异常

### 应急预案

| 异常情况 | 处理方案 |
|----------|----------|
| **玩家掉线** | 允许延迟行动/AI代管角色（限时） |
| **AI响应异常** | 使用预设备选文本/DM即兴发挥 |
| **剧情逻辑断裂** | 暂停修复/使用强制合并机制 |
| **玩家争执不下** | DM仲裁/投票表决/掷骰决定 |
| **时间严重超时** | 压缩后续环节/跳过次要事件 |
| **玩家情绪失控** | 暂停游戏，先冷静沟通 |

## 📊 常用速查表

### 检定难度参考
| 难度 | 成功率 | 适用场景 |
|------|--------|----------|
| 简单 | 75% | 常规行动 |
| 普通 | 55% | 需要技巧的行动 |
| 困难 | 35% | 高风险高回报 |
| 极难 | 15% | 逆转局面的行动 |
| 不可能 | 5% | 奇迹般的行动 |

### 关系数值含义
| 数值范围 | 关系状态 |
|----------|----------|
| 90-100 | 生死之交/深爱 |
| 70-89 | 可信赖的盟友/暧昧 |
| 50-69 | 表面友好/普通朋友 |
| 30-49 | 保持警惕/有所保留 |
| 10-29 | 对立/敌视 |
| 0-9 | 血海深仇 |

### BGM推荐列表
| 场景 | 推荐曲目风格 |
|------|-------------|
| 序章/回忆 | 凄美悲壮的古琴/箫 |
| 日常场景 | 轻快雅致的古筝/笛 |
| 紧张对峙 | 急促的鼓点/琵琶 |
| 浪漫场景 | 悠扬缠绵的琴箫合奏 |
| 战斗/冲突 | 激昂的打击乐/号角 |
