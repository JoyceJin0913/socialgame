## 🎭 什么是C2A2C互动剧本杀？

```
传统小说阅读：
  作者写 → 读者读 → （单向消费，无互动）
  
C2A2C互动剧本杀：
  原著基础 → AI动态改编 ↔ 多人角色扮演 ↔ 实时社交互动
                 ↑                                    ↓
            你的每一个选择 ←———————————→ 改变故事走向
```

### 核心体验
- ✅ **角色沉浸**：化身小说中的任意角色，感受他们的喜怒哀乐
- ✅ **剧情共创**：你的决策真正影响故事发展，没有固定的"正确答案"
- ✅ **社交乐趣**：与其他玩家实时互动、结盟、对抗、交易
- ✅ **无限可能**：每次游戏都是独特的体验，多重结局等你解锁

## 🎮 操作指引

### 基础操作

#### 1. 行动选择
每个回合，系统会提供可选行动：
```
【当前场景：闺房清晨】
┌─ 可用行动 ──────────────────┐
│ A. 🔍 检查身边物品           │
│ B. 👀 向窗外张望             │
│ C. 💬 叫丫鬟进来             │
│ D. 📝 查看当前状态           │
│ E. 💬 自由输入（自定义行动）  │
└───────────────────────────────┘

请选择 (A-E):
```

#### 2. 私聊功能
- 点击玩家头像 → 选择「私聊」
- 私聊内容**只有对方能看到**
- 可以用于：结盟、欺骗、交易、分享秘密

#### 3. 技能使用
```
【技能栏】
┌─────────────────────────────┐
│ 🔄 前世记忆    [剩余: 3次]  │

│ 🛡️ 姐弟情深    [被动]      │
│ 💫 扭转乾坤    [终极]      │
└─────────────────────────────┘

使用技能? (Y/N)
```

#### 4. 物品/秘密管理

## 💡 策略建议

### 根据你的角色定位...

#### 👱‍♀️ 如果你扮演庄寒雁（主角）
- **核心策略**：隐忍发育，伺机反击
- **优先目标**：收集证据、保护弟弟、识别可信盟友
- **技能使用**：合理使用「前世记忆」（只有3次！）
- **感情线**：可与傅云夕/赫连珡发展，但会影响结局
- ⚠️ **切记**：不要过早暴露重生身份！

#### 👑 如果你是傅云夕（男主）
- **核心策略**：运筹帷幄，暗中守护
- **优先目标**：调查父王死因、接近寒雁、打压敌人
- **技能使用**：「王府求娶」是终极技能，选对时机！
- **感情提示**：太强势会吓跑她，太被动会输给别人

#### 🦊 如果你是庄语山（反派）
- **核心策略**：伪装伪装再伪装
- **优先目标**：骗取信任、破坏寒雁、攀上高枝
- **技能使用**：「白莲花」是你的核心，善用它！
- **生存法则**：永远留一条后路

#### 🐍 如果你是周氏（大反派）
- **核心策略**：幕后操控，借刀杀人
- **优先目标**：巩固地位、除掉障碍（寒雁！）、安排女儿婚事
- **技能使用**：「毒礼相赠」要放长线钓大鱼
- ⚠️ **高风险高回报**：一旦暴露就是毁灭性的

#### ⚔️ 如果你扮演庄寒明（弟弟）
- **核心策略**：成长学习，保护姐姐
- **优先目标**：搜集证据、提升能力、不被坏人带坏
- **特殊优势**：因为年幼，大人会对你降低防备
- **终极目标**：从被保护者变成保护者！

#### 💙 如果你扮演汲蓝（忠仆）
- **核心策略**：辅助为主，关键时刻挺身而出
- **优先目标**：传递情报、保护主人、联络盟友
- **特殊价值**：你的信息网非常重要
- **终极时刻**：「关键证词」可以直接定罪反派！

## 🌐 社交玩法

### 阵营与联盟

游戏中你可以：

| 行动 | 效果 |
|------|------|
| 🤝 **结盟** | 与其他玩家组成临时/永久同盟 |
| 🔄 **背叛** | 出卖盟友换取利益（但有风险！） |
| 💰 **交易** | 用秘密/物品/技能交换资源 |
| 🗳️ **投票** | 参与集体决策（如放逐某人） |

### 关系数值系统
- 你的每个行动都会影响与其他角色的**关系值**
- 关系值高低决定了：
  - 能否结盟
  - 能否得知对方的秘密
  - 最终能否达成特定结局
  - 能否触发特殊剧情

## ❓ 常见问题

**Q: 我可以偏离原著设定吗？**
> A: 当然！这正是C2A2C的精髓。AI会根据你的选择实时改编剧情。

**Q: 不知道该怎么做怎么办？**
> A: 可以随时查看「状态面板」，或向系统请求 hints（会扣分哦）。

**Q: 可以和其他玩家商量吗？**
> A: 当然！私聊、公开讨论都可以，这也是游戏乐趣的一部分。但要小心——对方可能在骗你！

**Q: 死了还能继续玩吗？**
> A: 取决于具体情境。某些情况下可以"魂魄"形式存在观战，或在特殊条件下复活。

**Q: 这游戏适合什么人玩？**
> A: 
> - 小说爱好者（尤其是这本的粉丝）
> - 剧本杀爱好者
> - 喜欢角色扮演/cosplay的玩家
> - 社牛或想练社交的朋友
> - 想尝试新鲜事物的任何人
