# 🎭 《重生之贵女难求》C2A2C互动剧本杀

> **打破「读者」身份，化身故事主角——AI驱动的沉浸式社交推理游戏**

---

## ✨ 产品亮点

| 特性 | 说明 |
|------|------|
| 🎭 **角色代入** | 8位可玩角色，每个都有独特技能、秘密和命运 |
| 🤖 **AI动态改编** | 实时根据玩家选择改编剧情，每次游戏都是独特体验 |
| 👥 **多人社交** | 支持4-8人同时在线，结盟/对抗/交易/背叛 |
| 🔀 **多重结局** | 6种以上结局（True/Good/Normal/Bad/Dark/Hiden） |
| 💕 **CP系统** | 真实的感情线发展，影响最终结局 |

---

## 📖 故事梗概

**侯门嫡女庄寒雁**，前世被继母与庶姐设计陷害，新婚之夜饮毒酒而亡。

她重生回**十二岁那年**，带着前世的记忆与刻骨仇恨，誓要改写命运。

而这一次——**由你来决定她的命运走向。**

你是选择：
- 😇 **隐忍复仇**，步步为营让罪人付出代价？
- 🖤 **黑化之路**，以恶制恶成为新的恐惧？
- 💕 **追寻真爱**，在玄清王怀中找到救赎？
- ⚔️ **掌控全局**，权谋天下笑到最后？

**一切，由你选择。**

## 🎮 快速开始

### 方式一：体验演示（无需编码）
1. 打开 `player_handbook.md` 了解玩法
2. 使用 ChatGPT/Claude 作为GM
3. 将 `config/characters.json` 分配给各玩家
4. 按照 `scripts/story_scenes.json` 推进剧情

### 方式二：部署自建服务
详见 [DEPLOYMENT.md](./DEPLOYMENT.md)

## 📁 文件结构

```
C2A2C互动剧本杀/
├── README.md              # ← 你在这里
├── DEPLOYMENT.md          # 部署指南（3种方式）
├── dm_handbook.md         # DM主持人手册（机密）
├── player_handbook.md     # 玩家手册（公开）
│
├── config/
│   ├── game_config.json   # 游戏主配置（人数/阶段/规则）
│   └── characters.json    # 8个完整角色卡牌+技能树
│
├── content/               # 📚 **互动文游内容库**（核心产物）
│   ├── worldview.json     # 世界观（时代背景/权力体系/人物关系）
│   ├── prompt_development.md  # Prompt开发（世界观+5角色卡Prompt）
│   └── story_content.json    # 分幕剧情(HTML) + 互动节点(JSON)
│
├── scripts/
│   └── story_scenes.json  # 五幕详细剧本（50+场景/AI改编规则）
│
└── tech/
    ├── ai_engine.md       # AI动态改编引擎规则
    └── reference_impl.js  # 技术API接口定义
```

### 🗄️ 内容库字段规范

| 字段 | 文件 | 格式 | 说明 |
|------|------|------|------|
| 小说ID | `content/worldview.json` | JSON | `novelId: "chongsheng_gui_nv_001"` |
| 角色信息 | `content/prompt_development.md` | JSON+MD | 5名核心角色卡（姓名/身份/性格标签/核心矛盾/亲密度/技能/Prompt） |
| 分幕剧情 | `content/story_content.json` | HTML | 五幕沉浸式HTML叙事正文 |
| 互动节点 | `content/story_content.json` | JSON | 每幕2-3个决策点/社交点/CP事件（含选项-效果-后果链） |

## 🎬 五幕剧情

```
第一幕 · 前世噩梦 (30min)  →  序章夺命喜宴 + 重生觉醒
        ↓
第二幕 · 重生归来 (45min)  →  继母入门 / 茶楼初遇(CP) / 夜间行动
        ↓
第三幕 · 宅斗风云 (50min)  →  寿宴暗斗 / 马球争锋 / 证据收集
        ↓
第四幕 · 朝堂博弈 (45min)  →  宫宴风云 / 金銮决战(高潮)
        ↓
第五幕 · 命运终章 (30min)  →  结局演绎 + 复盘颁奖

总时长：约 3-4 小时
```

完整剧本见 [`scripts/story_scenes.json`](./scripts/story_scenes.json)

## 🏆 结局类型

| 结局 | 条件 | 情感色彩 |
|------|------|----------|
| 🌸 **True End** | 复仇成功+感情圆满 | 甜蜜治愈 |
| ✨ **Good End** | 大体顺利小有遗憾 | 温馨感人 |
| 🌙 **Normal End** | 平凡才是唯一答案 | 平淡真实 |
| 💔 **Bad End** | 计划失败重要之人逝去 | 悲壮催泪 |
| 🖤 **Dark End** | 选择黑化之路 | 爽快空虚 |
| 💋 **Harem End** | (隐藏) 全员好感满 | 沙雕彩蛋 |

## 🤖 AI引擎特性

- **实时叙事生成**：根据行动动态生成古风文本
- **分支剧情管理**：支持短期/中期/长期三级分支
- **多用户交互协议**：处理并发冲突与连锁反应
- **情绪状态追踪**：角色情绪影响可用选项与成功率
- **结局智能计算**：综合所有决策输出个性化结局

技术细节见 [`tech/ai_engine.md`](./tech/ai_engine.md)
